# 闲置换 - 二手交易市场

> 简单好用的二手交易平台，纯前端静态页面，数据存在浏览器本地存储，可以直接使用 GitHub Pages 部署。

## 功能特性

- ✨ **现代美观设计** - 渐变色、圆角、阴影、流畅动画，响应式适配手机/电脑
- 🔍 **搜索筛选** - 支持按关键词搜索，按分类筛选
- 📦 **发布商品** - 支持上传图片、填写价格、分类、成色、描述、位置
- 👀 **商品详情** - 点击查看完整详情，一键复制联系信息
- 💾 **本地存储** - 数据保存在浏览器 LocalStorage，刷新不丢失
- 🚀 **零后端** - 纯 HTML/CSS/JavaScript，不需要服务器，GitHub Pages 直接部署
- 📱 **响应式** - 完美适配手机、平板、桌面不同屏幕尺寸

## 页面预览

- 导航栏 - Logo、搜索、分类筛选、发布按钮
- 英雄区 - 展示平台介绍和统计数据
- 分类标签 - 快速切换不同分类
- 商品网格 - 瀑布流网格展示商品卡片
- 发布弹窗 - 表单发布新商品
- 详情弹窗 - 查看商品完整信息
- 页脚 - 导航链接

## 分类支持

- 数码电子
- 服饰鞋包
- 家居家具
- 书籍文具
- 运动健身
- 美妆护肤
- 其他

## 本地运行

直接打开 `index.html` 就能运行：

```bash
# Python 简易服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 部署到 GitHub Pages

本项目已经配置好 GitHub Actions 自动部署，推送到 GitHub 后自动发布。

## 技术栈

- 纯 HTML5 + CSS3 +  vanilla JavaScript
- 谷歌字体 Inter
- LocalStorage 数据存储
- GitHub Pages 托管

## 使用说明

1. **浏览商品** - 打开网站就能看到所有商品，点击分类筛选，搜索框搜索
2. **发布商品** - 点击「发布闲置」按钮，填写信息，上传图片，发布即可
3. **联系卖家** - 打开商品详情，点击「联系卖家」，商品信息自动复制到剪贴板
